PER FAR FUNZIONARE QUESTO CODICE E' IMPORTANTE SEGUIRE I SEGUENTI PASSAGGI:
1) INSTALLARE VISUAL STUDIO CODE E DOCKER DESKTOP
2) INSTALLARE IL WSL
3) APRIRE DOCKER DESKTOP
4) APRIRE IL TERMINALE DI DOCKER
5) ESEGUIRE QUESTO COMANDO: docker build -t esercizio_docker .
6) UNA VOLTA CHE IL PROGRAMMA HA FINITO IL BUILDING ESEGUIRE QUEST'ALTRO COMANDO PER
ATTIVARE IL SERVER docker run -d -p 3000:3000 --name ITS_container esercizio_docker
7) APRIRE IL FILE HTML/Index.html TRAMITE VSCode ATTRAVERSO live server
8) CLICCARE SU "Clicca qui"
9) TESTARE LA PAGINE DI LOGIN SCEGLIENDO TRA UNO DEI NOMI UTENTI CONTENUTI NEL FILE users.json CON
CORRISPONDENTE PASSWORD
10) SE TUTTO E' ANDATO A BUON FINE DOVREBBE APPARIRE NELLA PAGINA "Login successful! Token:" 
SEGUITO DAL TOKEN CORRISPONDENTE
